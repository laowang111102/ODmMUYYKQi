# 基于Tkinter的图书管理系统 python870

## 项目概述

本项目是基于Python编程语言的图书管理系统界面实现，使用Tkinter作为图形用户界面库，提供了一套简捷、实用的图书管理解决方案。

## 技术栈

- **编程语言**：Python
- **开发环境**：PyCharm/IDLE
- **界面库**：Tkinter

## 功能模块

- **用户模块**
  - 用户注册
  - 用户登录
  - 用户信息永久化存储

- **图书管理模块**
  - 图书增加
  - 图书删除
  - 图书修改
  - 图书查询
  - 还书功能
  - 图书信息永久化存储

## 系统角色

- 系统管理员：负责管理用户和图书信息
- 普通用户：可以进行图书查询、借阅、还书等操作

## 运行环境

- 操作系统：兼容Windows、Linux、Mac OS等
- Python版本：3.x

## 目录结构

```
项目名/
│
├── main.py       # 主程序入口
├── db/           # 数据库相关操作
│   └── db_handler.py
│
├── views/        # 视图层逻辑
│   ├── login.py
│   ├── register.py
│   ├── book_management.py
│   └── ...
│
└── models/       # 数据模型
    └── user.py
```

## 部署步骤

1. 安装Python环境（版本3.x）
2. 使用pip安装Tkinter（如果未预装）
3. 克隆或下载项目代码
4. 运行`main.py`启动程序

## 核心亮点

- 界面简洁，操作直观，适合快速入门与使用
- 用户信息与图书信息的永久化存储，数据安全有保障
- 基于Python开发，跨平台性能好，易于维护与扩展
- 功能模块清晰，方便后续功能升级与优化
